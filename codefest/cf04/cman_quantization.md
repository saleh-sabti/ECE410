## Task 5: S_bad = 0.01 Experiment

With S_bad = 0.01, the representable range is [-1.28, 1.27]. Four elements fall outside it and saturate.

| W | W / 0.01 | W_q (clamped) | W_deq | abs error |
|---|----------|---------------|-------|-----------|
| 2.10 | 210 | 127 | 1.27 | 0.83 |
| -1.88 | -188 | -128 | -1.28 | 0.60 |
| 1.55 | 155 | 127 | 1.27 | 0.28 |
| -2.31 | -231 | -128 | -1.28 | 1.03 |

The other 12 elements divide evenly by 0.01 so their quantization error is 0.

MAE = (0.83 + 0.60 + 0.28 + 1.03) / 16 = **0.171**

When S is too small, elements larger than S x 127 all saturate to the same clamped value, and the original magnitude is permanently lost.
