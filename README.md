**Spring 2026 | Portland State University**

**Student:** Saleh Sabti

## Tentative Project Topic
Hardware accelerator for real-time echo detection in voice signals.

A fixed-function chiplet that computes correlation between a far-end reference signal and a microphone input, producing a binary echo-present / echo-absent decision in hardware. The goal is to remove the detection burden from the CPU, reduce decision latency, and enable a downstream cancellation stage to activate only when needed.
